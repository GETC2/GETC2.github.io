Virtual Host Discovery

之前说过在针对Web进行渗透测试时，需要知道目标的Host Name。因为一个IP可能有多个Host，不知道对应的域名，就无法访问网站，那么渗透测试也就无从说起。

针对这个问题，之前通过DNS试图寻找目标的隐藏Host。其实我们还可以通过HTTPS证书来查找目标的隐藏域名。

实验说明

为了演示，我们只选择一个域名作为目标：sec542.com。

这里我们使用一个专门查询证书的网站：https:\\crt.sh。

这里的百分号是通配符，匹配任意的子域名。搜索结果如下：

可以看到，通过搜索出来的证书，我们可以发现目标域名的子域名。

除了这个网站，以下的网站也可以针对目标查询证书：

https://transparencyreport.google.com/https/certificates

不过第一个网站的搜索结果更为完备一些。
